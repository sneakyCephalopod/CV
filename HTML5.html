What I've learned with HTML5:

  1. Template:
    • Information for the browser on how to read document
    • Language for the browser
    • Head Section
    • Body Section
  2. Components in the Head Section
    1. meta tags: information for the browser
      • Name
      • http-equiv
      • charset
      • content
    2. Title: information displayed on the tab
  3. Components of the Body Section
    1. Headings
    2. Paragraphs
    3. Screen Reader prompts
      • Em: emphasis or italics
      • Strong: bold
      • Alt in image source
    4. Lists
      • Unorder (ul)
      • Ordered (ol)
        • Attributes:
        • Type
          • a - for lowercase letters
          • A - for upper case letters
          • i - for lowercase Roman numerals
          • I - for uppercase Roman numerals
          • Default or 1 for numbers
        • reversed
        • start: which starts where you tell it
    5. Loading Images
      • Image (img)
        • Attributes:
        • source (src) location of the images whether online or in website's folder
        • alternative text (alt): what a screen reader will read
        • Many more!
      • Use Photobucket to store images online for your website.
      • Use online cropping tools if needed
    6. Hypertext References (a href)
      • Website address
      • Attributes:
      • # + heading title will link to a place in the same document
      • Link a telephone number with "tel:5555555555"
      • Link an email address with "mailto:someone@gmail.com"
    7. Tables
      • table
      • thead
      • tbody
      • tr
      • td
      • tfoot
    8. Forms
      • form
        • class
        • action
        • method
      • label
      • input
        • Attributes:
        • type
        • id
        • name
      • textarea
